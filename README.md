# hotel-api

## Requirements
- PHP ^8.2
- Laravel ^11.0

## Installation
- 1. Clone Repo from GitHub
```bash
git clone https://github.com/MahmoudShalma/hotel-api.git
```
- 2. create a `.env` file
```bash
cd hotel-api && cp .env.example .env
```
- 3. install composer 
```bash
composer install
```
- 4. Import Postman 
```bash
attached file
```
